# Laravel REST API – Authentication & Roles

Backend API desarrollada con Laravel que implementa autenticación,
autorización por roles y operaciones CRUD.

## Tecnologías
- Laravel
- PHP
- MySQL / PostgreSQL
- Docker
- Sanctum (Auth)

## Funcionalidades
- Registro y login de usuarios
- Autenticación basada en tokens
- Roles y permisos
- CRUD de recursos
- Validaciones y manejo de errores

## Instalación
1. Clonar repositorio
2. Configurar .env
3. Ejecutar migraciones
4. Levantar con Docker
